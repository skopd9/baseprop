# User Types Guide

## Overview

PropertyFlow supports three distinct user types, each with different needs and workflows for managing rental properties. This guide explains each user type and how to make the most of the platform based on your role.

## üè† User Types

### 1. Direct Landlord (Self-Managing)
**Who:** Landlords who manage their properties entirely on their own  
**Also known as:** DIY landlord, self-managing landlord, hands-on landlord

#### Characteristics
- Find and vet tenants themselves
- Handle all tenant communications directly
- Manage all maintenance and repairs
- Track rent collection personally
- Handle all compliance requirements
- Deal with tenant issues directly

#### PropertyFlow Features Available
‚úÖ **Full Property Management**
- Add and manage properties
- Complete property details and financial tracking
- Upload and track all documents

‚úÖ **Complete Tenant Management**
- Add tenants with full details
- Track lease agreements
- Monitor rent payments
- Send rent reminders
- Manage tenant communications
- Handle deposit protection

‚úÖ **Compliance Tracking**
- Track all certificates
- Set renewal reminders
- Upload compliance documents
- Monitor expiry dates

‚úÖ **Maintenance & Repairs**
- Log repair requests
- Track contractor details
- Monitor repair status
- Record costs

‚úÖ **Financial Tracking**
- Record all expenses
- Track rent payments
- Monitor property performance
- Generate reports

#### Best Practices for Direct Landlords
1. **Set up automated reminders** for compliance renewals
2. **Keep detailed records** of all tenant communications
3. **Document everything** - inspections, repairs, issues
4. **Use templates** for common communications
5. **Stay organized** with digital copies of all documents
6. **Know your obligations** - review UK Compliance Guide regularly

#### When to Consider Using an Agent
- Portfolio grows beyond 5-10 properties
- Don't have time for tenant vetting
- Living far from your properties
- Facing difficult tenant situations
- Struggling with compliance tracking

### 2. Agent-Using Landlord
**Who:** Landlords who use letting agents for some or all property management  
**Also known as:** Portfolio landlord with agent support

#### Characteristics
- Use letting agent for tenant finding
- May use agent for rent collection
- May use agent for property management
- Retain oversight of investments
- Want visibility without day-to-day involvement

#### Common Agent Services Used
- **Tenant Find Only** - Agent finds tenant, you manage
- **Rent Collection** - Agent collects rent, you manage property
- **Full Management** - Agent handles everything

#### PropertyFlow Features Available
‚úÖ **Property Portfolio Overview**
- Track all properties in one place
- Monitor financial performance
- View agent-managed vs self-managed properties

‚úÖ **Agent Management**
- Record agent details
- Track agent fees and commissions
- Store agent contracts
- Contact information readily available

‚úÖ **Tenant Overview**
- View tenant information
- Monitor rent payment status
- Track lease dates
- View agent-found tenants

‚úÖ **Compliance Monitoring**
- View compliance status
- Track certificate expiry dates
- Ensure agent is keeping properties compliant

‚úÖ **Financial Overview**
- Monitor rent income
- Track agent fees
- Record property expenses
- Calculate net returns

‚úÖ **Selective Tenant Management**
- View details (agent updates)
- Monitor status
- Limited direct editing (depending on agreement)

#### PropertyFlow Setup for Agent-Using Landlords

1. **Add Your Agent(s)**
   - Go to Settings ‚Üí Agents
   - Add agent details and services provided
   - Record commission structure

2. **Link Properties to Agents**
   - Mark which properties are agent-managed
   - Specify which services agent provides

3. **Track Which Tenants Were Found by Agent**
   - Record "Found via Agent" when adding tenants
   - Useful for commission tracking

#### Working with Your Agent

**Clear Communication:**
- Define responsibilities clearly
- Set expectations for reporting
- Agree on communication frequency

**Use PropertyFlow for Oversight:**
- Request regular updates from agent
- Input data for your own tracking
- Monitor agent performance

**Document Everything:**
- Keep records of agent communications
- Store copies of certificates they obtain
- Track agent fees and expenses

#### When to Switch from Agent to Self-Managing
- Want more control over property
- Reduce costs (agent fees typically 10-15%)
- Have more time available
- Portfolio is small enough to manage
- Good experience with property management

#### When to Increase Agent Services
- Portfolio growing rapidly
- Living far from properties
- Want to free up personal time
- Dealing with problem tenants
- Facing legal issues

## üîÑ Switching Between User Types

PropertyFlow allows you to change your user type as your situation evolves:

### From Direct Landlord ‚Üí Agent-Using Landlord
1. Add your agent to the system
2. Link relevant properties to agent
3. Continue using PropertyFlow for oversight
4. Mark which management tasks agent handles

### From Agent-Using Landlord ‚Üí Direct Landlord
1. Update property status to self-managed
2. Take over all management functions
3. Start using all PropertyFlow features
4. Ensure smooth handover of documents/access

### From Direct Landlord ‚Üí Property Manager
1. Change user type to Property Manager
2. Add landlord/client information
3. Tag properties by owner
4. Set up multi-landlord reporting

## üéØ Choosing Your User Type

### Start as Direct Landlord if:
- You have 1-3 properties
- Properties are close to where you live
- You have time to manage properties
- You want maximum control
- You want to minimize costs

### Consider Agent-Using Landlord if:
- You have 4+ properties
- Properties are far from your location
- You have limited time
- You're new to landlording
- You want professional tenant vetting

### Work as Property Manager if:
- You manage properties professionally
- You manage for multiple owners
- You need bulk operation capabilities
- You provide property management services
- You have a letting agency

## üí° Tips for Success

### For All User Types
1. **Keep PropertyFlow updated** - Regular data entry ensures accuracy
2. **Set up reminders** - Never miss compliance deadlines
3. **Document everything** - Photos, notes, communications
4. **Review regularly** - Monthly review of all properties
5. **Stay informed** - Keep up with UK landlord legislation

### For Direct Landlords
- **Join landlord forums** for peer support
- **Build contractor relationships** for reliable service
- **Consider insurance** - landlord insurance is essential
- **Set boundaries** with tenants (response times, contact methods)

### For Agent-Using Landlords
- **Regular agent reviews** - Ensure they're performing well
- **Verify compliance** - Don't assume agent is handling everything
- **Understand your contract** - Know what you're paying for
- **Stay involved** - You're ultimately responsible

### For Property Managers
- **Professional indemnity insurance** is essential
- **Client communication** is key to retention
- **Systems and processes** make you more efficient
- **Keep learning** - Laws change, stay current
- **Build your reputation** - Good service leads to referrals
